Basic Usage

1. Upload css-animated-flexslider folder to your server and add the required files in your page (usually in the head section). Use the online build tool to optimize your slider build and create the main JS file (flexslider.js).

<!-- basic stylesheet -->
<link rel="stylesheet" href="css/flexslider.css">

<!-- Plugin requires jQuery 1.8+  -->
<!-- If you already have jQuery on your page, you shouldn't include it second time. -->
<script src='js/jquery-1.8.3.min.js'></script>

<!-- Main slider JS script file --> 
<!-- Create it with slider online build tool for better performance. -->
<script src="js/jquery.royalslider.min.js"></script>

2. Add the slider HTML code to the body section. Every child element will be treated as a slide. Type of tag doesn't matter, it can be an unordered list, just list of divs, whatever. You can learn more about slide types in next sections of documentation.

<div>
	<!-- simple image slide -->
	<img src="image.png" />

	<!-- HTML content slide -->
	<p>Content goes here</p>

	<!-- image and content -->
	<div>
		<img src="image.png" />
		<p>Some content after...</p>
	</div>

	<!-- HTML content -->
	<div>
		Put any HTML content here
	</div>

</div>

3. Initialize the slider usually at the end of the body section.

<script>
	$('.flexslider').flexslider({
		animation: "slide",
		controlNav: false
	});
</script>

4. Apply size to slider via CSS (optional). If you want slider to scale height based on width, take a look at the autoScaleSlider option.

.flexslider {
	width: 790px;
	margin: 0 auto;
	box-shadow: none;
}
